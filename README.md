# react-daisy

Deadly simple template to start react with vite + tailwindcss + daisyUI + Heroicons!

## installation

```
npm install -g degit
degit harrysaputra/react-daisy my-new-project
```

or

```
npx degit harrysaputra/react-daisy my-new-project
```

## How to use
```
cd my-new-project
yarn
yarn dev
```
